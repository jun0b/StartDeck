# 🌟 StartDeck
**新しいタブを、カスタマイズ可能なウィジェットベースのホームダッシュボードに変える拡張機能です。**

---

## ✨ 特徴
* **カスタマイズ可能なダッシュボード:** ウィジェットを自由に配置・リサイズし、自分だけのレイアウトを作成・編集できます。
* **多彩なウィジェット:**
  * **カレンダー / ミニカレンダー:** スケジュールの管理・確認
  * **メモ / タスク:** シンプルなToDoリストやテキスト保存
  * **時計 / 天気:** 一目で時間帯と気象情報を確認
  * **ブックマーク:** 頻繁にアクセスするサイトのショートカット
  * **RSS / ニュース:** 最新のフィード情報を自動で取得
  * **メディアコントローラー:** 他のタブで再生中のコンテンツ（YouTubeなど）をダッシュボードから操作
  * **パスワード生成:** 安全なパスワードをカスタマイズして生成・コピー
  * **株価 / 検索 / Web埋め込み:** 必要な情報を柔軟に統合
* **美しいUIとテーマ対応:** ライトモードやダークモードなど、見やすく洗練されたデザインで作業への没入感を高めます。
* **ローカルで完結:** デフォルトの機能や設定、保存データはブラウザ内に保管され、アカウント登録は不要ですぐに利用可能です。

---

## 📥 インストール方法
この拡張機能は、Chromeウェブストア審査前の「開発者モード」での利用を想定しています。

### 📦 1. ファイルのダウンロードと準備
1. 最新のソースコード（ZIPファイル）をダウンロードし、解凍（展開）します。
2. **【重要】** ファイルを解凍してできた **フォルダ全体** を使います。

### ⚙️ 2. Chromeでの読み込み手順
1. お使いのブラウザ（Google Chrome推奨）を開きます。
2. アドレスバーに **`chrome://extensions`** と入力して開きます。
3. 画面右上の **「デベロッパーモード」** (Developer Mode) をONにします。
4. 左上の **「パッケージ化されていない拡張機能を読み込む」** ボタンをクリックします。
5. 手順1で解凍した **フォルダ全体** を選択して読み込みます。

### 🚀 3. 使い方
1. ブラウザで新しいタブを開きます。
2. 自動的にStartDeckのダッシュボードが表示されます。
3. 追加メニュー等からウィジェットを配置し、目的に沿ってカスタマイズしてください。

## 🙏 謝辞 (Credits)
本拡張機能は、以下の素晴らしいオープンデータやAPIによって支えられています。
各機能で利用している外部サービスに深く敬意を表します。

* **Weather Data:** [Open-Meteo](https://open-meteo.com/) (APIキー不要のオープンソース天気API)
* **Images:** [Unsplash](https://unsplash.com/) (フォトグラファーコミュニティ)
* **Astronomy Data:** [NASA APOD API](https://api.nasa.gov/) (今日の天文写真の取得)
* **Stock Data:** [Yahoo Finance API](https://finance.yahoo.com/) (株価およびチャート情報の取得)

※その他の外部データ（RSS等）は、Chrome拡張機能特有の仕組みを利用して安全にバックグラウンドから直接取得しています。
